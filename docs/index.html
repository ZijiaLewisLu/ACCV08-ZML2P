Facial expression dataset with Multiple Emotions
(FaceME)
- Zero-Shot Facial Expression Recognition with Multi-Label Label Propagation, ACCV 2018

FaceME dataset is a Multi-Label Facial Experssion Dataset collected by the Affective Computing Group of VIPL Lab.
It contains 10,062 images annotated with 85 affective labels. The purpose of the dataset is to describe facial experssions in a more diversified and detailed fashion. The 85 affective labels cover the main aspects of the information that facial expressions can convey. Specifically, they come from four categories:

  1. Emotions: Happy, Shocked, ...
  2. Interal Feelings: Apathy, Distracted, ...
  3. Actions: Crying, Scream, ...
  4. Health Status: Cough, Sick, ...
Each image is assigned to 3-4 annotators to select the labels best matching it.
